== Installation ==

=== Plugin Install ===

To install Typus as a plugin:

----
$ script/rails plugin install git://github.com/fesplugas/typus.git
----

NOTE: To re-install the plugin use the +--force+ option.

=== Gem Install ===

To install **Typus** as a +gem+, edit the application +Gemfile+ and 
add the following line.

----
gem "typus", :git => "git://github.com/fesplugas/typus.git"
----

Install the gem using the bundler:

----
$ bundle install
----

=== Running the Generator ===

Once the plugin or gem is installed you need to run a generator to 
create some required files. **Typus** files and migrate 
your database. A +typus_users+ table will be created, this is 
where **Typus** users are stored.

----
$ script/rails generate typus
$ rake db:migrate
----

Start the application server, go to http://0.0.0.0:3000/admin and 
follow the instructions.

=== Rake Tasks ===

To see a list of Typus related tasks run ...

----
$ rake -T typus
----
